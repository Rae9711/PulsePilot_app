## 🚀 How to Run the Persona Demos

### Quick Start Guide

1. **Open FitForecast**: http://localhost:5174/login

2. **Choose a Persona**:
   - Click demo button for desired user
   - Password is pre-filled: `password123`
   - Click "Login"

3. **Run the full-use flow**:

  **Dashboard** (`/`):
  - Show current context and immediate priorities
  - Explain why this user's recommendations differ from others

   **History** (`/history`):
  - Ground the story in real logs and feelings
  - Point to concrete examples backing current guidance

  **Analytics** (`/trends`):
  - Show trend chart + weight chart
  - Highlight top correlation and top next action cards

  **Log** (`/log`):
  - Add one fresh, persona-consistent entry
  - Capture pre/post feelings
  - Show AI advice generated from that new entry + today's context

4. **Compare Personas**:
   - Logout
   - Login as different user
   - Compare trends and insights
   - Notice different recommendations

```bash
  -H "Authorization: Bearer YOUR_TOKEN" | jq 'length'

```

